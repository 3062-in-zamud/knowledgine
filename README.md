# knowledgine

Developer Knowledge Infrastructure - Extract structured knowledge from your codebase for AI coding tools.

## Development

```bash
pnpm install
pnpm run verify
```

## Packages

- `@knowledgine/core` - Core types and utilities
- `@knowledgine/mcp-server` - MCP server for AI coding tools
- `@knowledgine/cli` - Command-line interface
